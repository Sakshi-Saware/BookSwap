Users can donate, borrow, or list books, with Firebase used for authentication and user data.

---

##  Features
- Student-to-student book exchange
- Add, remove, and browse book listings
- Real-time Firebase database updates
- Clean UI with responsive components
- Book category filtering
- Contact / request module (if included)

---

##  Tech Stack
**Frontend:** React, JavaScript, Tailwind / CSS  
**Backend:** Firebase Realtime DB / Firestore  
**Hosting:** Vercel
